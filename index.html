Cheatsheet

die wichtigsten Terminalbefehle

  • pwd
  • cd
  • ls
  • mkdir
  • touch
  • man

pwd ⇢ Zeigt das aktuelle Arbeitsverzeichnis an

cd ⇢ Verzeichnis wechseln

ls ⇢ Auflistung von Verzeichnissen (Inhalt eines Verzeichnis anzeigen)

mkdir ⇢ Ein Verzeichnis / Ordner erstellen

touch ⇢ Eine Datei erstellen

man- ⇢ Manual des nachfolgenden Befehles z.B. -ls
